## Hailley Griffis Weekly Planning Routine
Hailley Griffis uses a **weekly planning ritual** combining reflection and AI-processed voice notes for pattern recognition, optimized for work, personal goals, and side projects in Notion.[1]  
**Steps**:  
- Review last week's priorities: finished, carried over, or changed?  
- Reflect: what went well, improvements needed, changes for next week.  
- Process voice note transcripts with AI for insights (optional for verbal processors).  
- Link Notion databases for projects/goals; set 1-2 priorities per category (work, personal, side-projects).  
**Time**: Few minutes; done digitally in Notion or notebook.  
**Source**: http://www.hailleygriffis.com/weekly-planning/

## Alastair Weekly Log (ADHD-Focused)
The **Alastair Weekly Log** is a simple, flexible paper/digital template using dots to assign tasks to days (M-T-W-Th-F-S-Su), combating ADHD time blindness with 5-10 minute setup.[2]  
**Steps to start**:  
- List tasks on one page half (one per line).  
- Add day columns on the other half.  
- Place dots next to tasks for planned days.  
- Review/adjust weekly as plans shift.  
**Advantages over traditional planners**: Looser structure, visual snapshot, easy pivots; avoids rigid time blocks. Free template at imbusybeingawesome.com/weekly.  
**Source**: https://imbusybeingawesome.com/adhd-weekly-planning/

## Reflection.app Self-Reflection Guidelines
Reflection.app recommends **weekly self-reflection** starting at 5-10 minutes, focusing on one key weekly event via journaling highlights/lowlights for clarity on goals and improvements.[3]  
**Practice tips**:  
- Write weekly highs/lows; reflect on monthly/yearly/five-year goals, life changes, improvements.  
- Build habit: Link to routines like coffee/bedtime; start at 3-5 minutes with reminders.  
- Scale to daily as comfortable; use AI apps for personalized insights.  
**Source**: https://www.reflection.app/blog/self-reflection-101-what-is-self-reflection-why-is-reflection-important
