## ZSA Moonlander (Mark I and 2)
Fully split ortholinear ergonomic keyboard with hot-swappable switches, adjustable tenting, and thumb clusters ideal for programming customization like VS Code mappings.[2][3][1]

- **Ergonomic Design**: Space-age design reduces wrist extension from 22° to 9° via adjustable palm rests; supports split at shoulder width, columnar staggering, and independent splay/tenting/height per half to minimize ulnar deviation and pronation.[1][2][5][3]
- **Switch Recommendations**: Hot-swappable Cherry MX or Kailh options; users report no missed keypresses (e.g., Esc in Vim) unlike low-actuation alternatives; per-key RGB lighting.[3]
- **Tent/Tilt Options**: Highly adjustable tenting and tilting for everyday use; web-based Oryx configurator for tweaks; scored 8.39/10 in 2026 review for customization.[3][2]
- **Programming Notes**: ZMK firmware enables context-aware macros (e.g., Ctrl+Shift+P for VS Code Command Palette only when focused); 42-day battery life on Bluetooth.[1]
