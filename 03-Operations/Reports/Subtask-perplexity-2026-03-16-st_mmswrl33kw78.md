## Mail Sorting Workflows
- **Anytime Mailbox** enables remote mail sorting via a secure web portal or app, where users request scanning of envelopes (front/back), open and scan contents, then choose actions like forward, shred, or recycle. Workflow: Receive mail at one of 2,500+ global locations → view scans → select disposition digitally → staff executes. Supports check deposits at select sites for direct banking[1].
- **ButterflyMX Package Room** for multifamily/home setups uses staff-logged workflows in software like Notifii or PackageX: Carrier delivers → staff scans/logs package → notifies resident via app → resident retrieves post-hours via PIN. Audit logs track all entries for verification[2].

## Digital Scanning Processes

## Identity Theft Prevention
- **Anytime Mailbox** shreds sensitive mail on request (promotes security/environmental responsibility); uses real street addresses without revealing home info, akin to PO Boxes/USPS Smart Lockers[1][3].
- **ButterflyMX** employs surveillance cameras, audit logs (records every entry), and revocable PINs to prevent unauthorized access/theft[2].
- **Smart Lockers** ensure packages accessible only via unique recipient codes, mirroring USPS PO Box privacy; reduces mailroom mishandling[3].
