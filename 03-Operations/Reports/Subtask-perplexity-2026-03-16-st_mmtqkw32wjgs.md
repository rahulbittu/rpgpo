## Finding 1: Submission Processes for Architectural Review Boards
- Standardized online forms or platforms streamline submissions; e.g., AIA Minnesota requires electronic submission via their awards platform with anonymous materials (images, descriptions) and 3-4 letters of recommendation, due weeks before review[1].
- Homeowners Associations (HOAs) mandate pre-application reviews and submissions 3 weeks prior to meetings; Venetian Isles HOA assigns primary/secondary reviewers immediately upon submission[2][5].
- Texas Property Code (Sections 209.00505-209.00507) requires written denial notices via certified mail or email, specifying reasons and required changes[4].
Source: https://www.aia-mn.org/get-involved/aia-minnesota-awards/architect-distinction-award/[1]; https://www.cityofandersonsc.com/event/board-of-architectural-review-2/2026-05-20/[2]; https://guides.sll.texas.gov/property-owners-associations/ccrs[4]; https://www.ourvi.org/architectural-review.html[5]

## Finding 2: Review Criteria and Common Pitfalls
- Criteria focus on compliance with guidelines (design excellence, practice leadership); consistent application avoids disputes—pitfalls include poor communication, inconsistent decisions, delays beyond deadlines (e.g., auto-approval if not denied timely per Ohio Rev. Code §5311.081(C))[3][6].
- HOAs review for architectural/structural changes, new construction, fencing; must provide detailed written denials if issues like missing info or non-compliance[6].
- Pitfalls: Non-uniform criteria, ignoring due process (written notice + hearing before fines), board members reviewing own applications[3][4][6].
Source: https://www.williams-strohm.com/insights/architectural-enforcement-important-factors-to-keep-in-mind[3]; https://rmwbh.com/acc-basics/[6]; https://guides.sll.texas.gov/property-owners-associations/ccrs[4]

## Finding 3: Decision Authority and Appeals
- Boards/Committees hold final say, issuing Certificates of Appropriateness post-approval; e.g., City of Anderson SC Board decides at scheduled meetings[2].
- Texas law mandates 30-day appeal window to full board of directors for hearings to affirm/modify/reverse decisions[4].
- University of Washington Reference Architecture Review Board arbitrates implementation issues; ServiceNow uses governance for standards/guardrails[7][8].
- HOAs may deny for delinquencies if documents allow; variances only if explicitly authorized[6].

## Finding 4: Strategies for Effective Boards and Lightweight vs Formal Tracks
- Best practices: Clear timelines, standardized templates, uniform enforcement, accessible info to prevent lawsuits (Ohio Rev. Code §5311.19(A))[3].
- Separate ACC from board to avoid conflicts (no current board members/spouses); solicit volunteers first[6].
- Lightweight: Immediate reviewer assignment for simple projects (Venetian Isles)[5]; Formal: Scheduled meetings with pre-submissions 3 weeks ahead (Anderson SC), appeals processes[2][4].
- No explicit tech/software ARB examples in recent results; HOA processes emphasize smooth digital submissions over rigid boards[1][3][5].
Source: https://www.williams-strohm.com/insights/architectural-enforcement-important-factors-to-keep-in-mind[3]; https://rmwbh.com/acc-basics/[6]; https://www.ourvi.org/architectural-review.html[5]; https://www.cityofandersonsc.com/event/board-of-architectural-review-2/2026-05-20/[2]

## Actionable Next Steps
- **Draft templates**: Create standardized submission form with checklists for criteria (e.g., adapt AIA anonymity rules); test with 1-week turnaround for lightweight track[1][3].
- **Set timelines**: Implement 3-week pre-meeting submissions and 30-day appeals per Texas model; auto-approve if no response in 45 days[2][4].
- **Build committee**: Recruit 3-5 non-board volunteers; assign primary/secondary reviewers for speed[5][6].
- **Pilot lightweight track**: For minor changes (paint/fencing), use auto-assignment; escalate formal for new builds[2][5].
- **Monitor pitfalls**: Log all decisions for consistency audits quarterly; add due process notices[3].
