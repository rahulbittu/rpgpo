# RPGPO Governance Layers

## Layer 1: Identity
Who RPGPO is and what it exists to do.

## Layer 2: Constitution
Permanent values, principles, and non-negotiables.

## Layer 3: Operational Governance
Board roles, approval rules, memory rules, access rules, logging rules.

## Layer 4: Domain Governance
TopRanker rules, career rules, finance rules, creative rules, etc.

## Layer 5: Runtime Execution
Current task routing, agent runs, approvals, outputs.

## Layer 6: Audit and Review
Logs, dashboards, daily/weekly reviews, failure analysis.
