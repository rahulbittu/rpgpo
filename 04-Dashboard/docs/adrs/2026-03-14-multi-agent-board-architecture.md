# ADR: Multi-agent board architecture

**Status:** accepted
**Date:** 2026-03-14
**Modules:** board, conversations, agents

## Context
GPO needs governed multi-agent collaboration

## Decision
Use structured conversation fabric with privacy-scoped turns

## Consequences
Agents can collaborate without uncontrolled free chat

## Alternatives Considered
- Single-agent deliberation
- Unstructured agent chat
