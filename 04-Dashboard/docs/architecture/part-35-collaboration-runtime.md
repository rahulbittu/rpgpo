# Part 35: Multi-Agent Collaboration Runtime + Negotiation Protocols

## Overview
Structured agent-to-agent collaboration with negotiation protocols and consensus computation.

## Architecture
```
Collaboration Runtime → Sessions with proposals, turns, votes
Negotiation Protocols → 7 protocol types, structured resolution
Agent Consensus → Vote computation, dissent preservation
```

## Key Principle
Disagreements are preserved, not erased. Unresolved conflicts escalate, never silently drop.
