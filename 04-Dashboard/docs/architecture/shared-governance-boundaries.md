# Shared Governance Boundaries

## Default Boundaries

| Artifact Type | Sharing | Redaction | Approval |
|--------------|---------|-----------|----------|
| context | No | Required | Required |
| governance | Yes | No | No |
| override | No | Required | Required |
| exception | No | Required | Required |
| tuning | Yes | Required | Required |
| provider_fit | Yes | No | No |
| execution | No | Required | Required |
| promotion | No | Required | Required |
| pattern | Yes | Required | Required |

Provider heuristics and governance rules are freely shareable. Everything else requires redaction and/or approval.
