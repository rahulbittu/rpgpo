# GPO Operating Doctrine

## Core Principles

1. **Reliability beats autonomy theater.** GPO should work predictably. A system that works 95% of the time on understood tasks beats one that impresses 50% of the time on ambitious ones.

2. **Only automate what is understood.** If the system doesn't understand a workflow well enough to predict the outcome, it should assist rather than execute autonomously.

3. **AI should amplify understanding, not replace it.** GPO helps the operator think better and work faster — it doesn't make decisions the operator can't review.

4. **Predictability is a product feature.** The operator should know what GPO will do before it does it. Surprises erode trust.

5. **Governed execution beats agent chaos.** Every action that matters goes through deliberation, routing, and approval gates. No uncontrolled side effects.

6. **Human judgment stays where risk or ambiguity is high.** GPO escalates when uncertain. Auto-approval is earned through consistent low-risk performance, not assumed.

7. **Memory should reduce friction, not create false certainty.** Learning from operator behavior is good. Overconfidently applying stale or weak signals is dangerous.

8. **Hunt for why it fails, not just proof it works.** Evidence systems should catch problems, not inflate scores.

9. **Users want pain relief and dependable outcomes, not hype.** GPO solves real problems — career research, financial planning, writing, scheduling — not "AI agent" theater.

10. **Fewer stronger systems beat many noisy agents.** One well-tuned pipeline (Perplexity → OpenAI → deliverable) beats ten competing agents with no coordination.

## What GPO Is

- A privacy-first AI operating system for one operator
- 15 specialized engines with Board of AI deliberation
- A governed execution pipeline with approval gates
- A behavior learning system that gets smarter over time
- A local-first system with no data leaving the operator's control

## What GPO Is Not

- Not a public AI agent marketplace
- Not an autonomy-first system
- Not a replacement for human judgment
- Not a "one-click automate everything" tool
- Not an agent casino with uncontrolled side effects
