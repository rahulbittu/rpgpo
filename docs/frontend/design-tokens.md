# GPO Design Tokens

## Spacing

```css
--sp-2: 2px;
--sp-4: 4px;
--sp-8: 8px;
--sp-12: 12px;
--sp-16: 16px;

```

## Radii

```css
--r-sm: 4px;    /* badges, small elements */
--r-md: 6px;    /* cards, panels */
--r-lg: 8px;    /* modals, large surfaces */
```

## Surfaces

```css
--bg-base: #0a0d15;       /* page background */
--bg-surface: #111520;     /* card/panel */
--bg-elevated: #181d2a;    /* hover, active, dropdown */
--bg-inset: #0d1018;       /* inset areas, code blocks */
```
