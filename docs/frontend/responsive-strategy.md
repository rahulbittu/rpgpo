# GPO Responsive Strategy

## Breakpoints

| Range | Layout | Sidebar | Content |
|---|---|---|---|
| >1200px | Desktop | Full sidebar (200px) | Content with max-width |
| 768-1200px | Tablet | Icon-only sidebar (56px) | Full width |
| <768px | Phone | Bottom tab bar (56px) | Full width, stacked |

## Phone Navigation

Bottom tab bar with 5 primary items:
- Home
- Work
- Approvals
- Engines
- More (Activity + Settings)

## Grid Behavior

| Desktop | Tablet | Phone |
|---|---|---|
| 2-column grids | 2-column | 1-column |
| 3-column grids | 2-column | 1-column |
| 4-column metrics | 2x2 | 2x2 |

## Touch Targets

Minimum 44px height for all interactive elements on phone/tablet.
